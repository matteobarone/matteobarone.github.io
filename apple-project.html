Project goals

Location

Individual work

Brief

The project goal was to create a new digital experience that focuses on transmitting all the values and qualities of the product or service. The chosen brand is Apple, the product is the "iPod touch" and in particular with the "Music" app.

The result is a landing page that present the product in an emotional way, using different web technologies.

Before the final result, the project passed throw different stages: form the brainstorming to the prototyping.

Three main secitons

The page is basically divided in three main secitons:

- Video where the page wants to communicate the emotions of using the product.

- Intro with the title, description of the product and call to action to download the Music app or buy the iPod.

- The last part, the Features of the iPod, more technical, and less important for the purpose of the page.

The project was developed with a middle/high-fidelity prototype, using HTML and CSS, Javascript and a video.
